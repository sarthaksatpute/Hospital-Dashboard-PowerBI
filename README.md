## Project Overview

This project is a Hospital Performance Dashboard built using Power BI. 
It analyzes hospital quality across multiple domains such as Safety, Mortality, Readmission, and Patient Experience.

## Key Features

- Designed a Star Schema data model for efficient analysis
- Created advanced DAX measures for performance evaluation
- Implemented What-if analysis (Rating Threshold & Improvement Parameter) for scenario simulation
- Built interactive visuals including KPI cards, ranking, and trend analysis

## Tools Used

- Microsoft Power BI
- DAX (Data Analysis Expressions)
