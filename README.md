# 🏥 Hospital Performance Dashboard (Power BI)

## 📊 Project Overview

This project presents a **Hospital Performance Dashboard** built using Power BI.  
It analyzes hospital quality across key domains such as **Safety, Mortality, Readmission, and Patient Experience**, enabling data-driven insights and decision-making.

---

## 🚀 Key Features

- ⭐ Designed a **Star Schema Data Model** for optimized performance and scalability  
- 📈 Developed **Advanced DAX Measures** for in-depth performance analysis  
- 🎯 Implemented **What-If Analysis** (Rating Threshold & Improvement Parameter) for scenario simulation  
- 📊 Created **Interactive Visuals** including KPI cards, rankings, and trend analysis  
- 🔍 Enabled dynamic filtering and drill-down for better insights  

---

## 🛠️ Tools & Technologies

- 🟡 **Microsoft Power BI**  
- 📐 **DAX (Data Analysis Expressions)**  

---

## 💡 Insights Delivered

- Identify **high-performing hospitals** based on rating thresholds  
- Analyze performance across multiple healthcare domains  
- Simulate improvements using **What-if scenarios**  
- Support **data-driven decision-making** for stakeholders
